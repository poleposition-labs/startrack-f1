# Contributing to StarTrack F1

Thank you for your interest in contributing to StarTrack F1! We are building an open-source, community-driven motorsport analytics platform.

## Legal & Branding Guidelines (CRITICAL)

To maintain the legal safety of this project, all contributors must adhere to the following rules:

1. **No Official F1 Branding**: Do not use official Formula 1 logos, fonts, or trademarks.
2. **No Proprietary Data**: Do not import or use official live timing data or licensed datasets. Use only open or synthetic data.
3. **Descriptive Use Only**: You may use terms like "F1-style" or "Formula 1-style" to describe physics or mechanics, but do not imply official affiliation.

## How to Contribute

1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Development Standards

- **Code Style**: formatting via Prettier (frontend) and Black (backend).
- **Testing**: Ensure all new physics logic has unit tests.
- **Documentation**: Update documentation for any new modules.

## Reporting Issues

Please use the GitHub Issues tab to report bugs or suggest features.
